Container Packing Optimizer

How your order goes from Excel file to a packed container plan

📋
Input — Your Order File
An Excel spreadsheet listing each pallet type, its dimensions, and how many you've ordered
1
🔍
Read & Understand the Order
The file is scanned to find the relevant columns — even if they have slightly different names or blank rows at the top. Each row becomes a pallet type with its size (length × width × height) and quantity. Loose boxes labelled NP are separated out for later.
💡 Accepts both comma and dot decimals, flexible column names, and auto-skips any unrecognisable rows.
2
📦
Group Pallets into Physical Rows
Pallets are grouped into the rows they'll physically form inside a container. A row is a set of same-type pallets stacked and placed side-by-side across the container's 235 cm width. The number of pallets per row depends on their footprint and how many can be stacked safely.
⚠️ Your order quantity must be an exact multiple of each row's pallet count. If not, the tool tells you exactly how many pallets to add.
3
🧠
Work Out the Best Loading Order
A mathematical optimiser tests thousands of combinations and picks the arrangement that fits the most pallets into each container. It obeys three physical rules: the total length of rows must fit inside the container, the weight must stay under the limit, and taller rows must go at the back so everything can pass through the door.
📏 Max length 1 203 cm ⚖️ Max weight 18 000 kg 🚪 Door opening 250 cm ↕️ Tall rows loaded from rear
4
🗃️
Fit Loose Boxes into the Gaps
Once the pallet rows are placed, any remaining space at the back of the container is filled with the loose (NP) boxes from your order. The tool tries every orientation for each box type to find the densest fit, stacking multiple box types in the same section rather than dedicating a separate section to each.
📌 Boxes go in the tail space only — they can't be placed on top of pallets.
5
✅
Check Everything Adds Up
Before producing any output, the result is cross-checked: every pallet from your order must appear exactly once, no container can be over-length or over-weight, and the door-height rule must hold. Any discrepancy stops the process and reports a clear error.
🔎 8 automatic checks — if all pass, the plan is guaranteed to be physically valid.
6
💡
Suggest What Else Could Fit
The tool looks at any unused space — the gap after the last pallet row, and the headroom above shorter rows — and recommends additional pallets or boxes you could add to your order to make use of that space. Suggestions are proportional to what you're already ordering.
📈 Two zones checked per container: tail space and above-row headroom.
7
📊
Generate the Report
A formatted Excel report is created with four sheets: a summary overview, a detailed per-container breakdown, a colour-coded packing layout grid, and the fill recommendations. A 3-D image of each container is also rendered showing exactly where every row sits.
📥 The report downloads automatically in your browser.
📤
Output — Your Packing Plan
Excel report · 3-D container images · Fill recommendations · Warnings & errors
